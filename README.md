# Things_Reminders_ReactNative
Things + Reminders App using React-Native in place of Flutter and Dart.

# Goal
The Goal of this repo is to reproduce and upgrade my existing Flutter/Dart Things + Reminders app by using React Native and Typescript

# Process
Leverage Expo, Expo Router, Typescript / Javascript, and React Native to build a mobile app for iOS and Android.

# This is a public repo that is for educational purposes, and will be using mock data until release.

# Once released, this repo will turn private, leveraging Firebase for data storage
